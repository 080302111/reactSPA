

### Target
> 使用React技术栈开发SPA.

[效果展示](https://muyunyun.github.io/react-antd-demo)

### Usage
```
本地运行
yarn install or npm install
yarn start or npm start
```

### Tech Stack || Features
* UI库：React & React-Dom
* UI组件：Antd(2.9x)
* 路由：React-Router(4.x) & History
* 框架：Reflux(待实现)
* JS：ES6
* 样式：Less
* 图标：Antd自带/FontAwesome
* 动画：Animate.css
* 与后台通信：Fetch(待实现)
* 日期处理：Moment(待实现)
* 假数据模拟：MockJS(待实现)
* 打包构建：Babel Webpack
* 包管理：Yarn || Npm
